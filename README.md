yometab
=====================

なにこれ
---
mikutterに嫁を表示するタブを追加するプラグインです。新規ペインを立てて嫁を表示することで精神をいつでも安定させられます。

つかいかた
---
1. プラグインをインストールします
````
mkdir ~/.mikutter/plugin ; git clone https://github.com/Akkiesoft/yometab ~/.mikutter/plugin/yometab
````
2. 画像URLとサイズなどを設定して、mikutterを再起動します。
3. 嫁タブでさわやかておくれライフ
